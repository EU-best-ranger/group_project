Security

Security tip 1 - In order to ensure that we have a secure network we can make sure that all the students and members of staff understand that you cannot share accounts as that increases the danger of being hacked.

Security tip 2 - we can also ensure that we have proper password security, meaning that we regularly change passwords in order to protect the accounts. This also involves testing the strength of these passwords using specific websites.

Security tip 3 - We also have the option of limiting how many people have accounts and how much access these accounts can have, for example you could restrict wifi use for some accounts to limit risk of infection with malware.

Security tip 4 - Finally we could use two factor authentication systems which makes hacking accounts much harder.
